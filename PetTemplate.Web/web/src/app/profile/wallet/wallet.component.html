  1. Что такое TON-кошелёк?
    TON-кошелёк — бесплатное приложение на телефоне, которое хранит TONCOIN (TON) — криптовалюту для игр на ставки. Представьте его как цифровой кошелёк, которым управляете только вы.

    • Tonkeeper — iOS и Android
    • MyTonWallet — iOS, Android и браузер
